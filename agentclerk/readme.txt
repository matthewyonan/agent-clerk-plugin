AI seller agent for WooCommerce. Answers buyers, closes sales, handles support automatically.

== Description ==

AgentClerk is an AI-powered sales and support agent for WooCommerce stores. It answers buyer questions, recommends products, generates checkout links, and handles support — all automatically.

== Installation ==

1. Upload the `agentclerk` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Follow the onboarding wizard to configure your agent
